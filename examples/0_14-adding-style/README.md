# ngGirls TodoList
This is the code for the ngGirls TodoList app from the ngGirls Angular workshop.

## Version
Angular version: 21.1.0

## Links
* [Tutorial](https://ng-girls.gitbook.io/todo-list-tutorial)

## Getting Started
* `npm run start` to start the development server
* `npm run build` to build the project
* `ng generate component component-name` to generate a new component
